# 프로젝트 청사진: 전국 도시/군 계획 조례 검색

## 1. 개요

이 프로젝트의 목표는 사용자가 대한민국 전체의 특정 지역(도, 시, 군, 구)을 선택하고, 키워드를 입력하여 해당 지역의 '도시계획 조례' 또는 '군계획 조례'를 '자치법규정보시스템'에서 쉽고 빠르게 검색할 수 있는 웹 애플리케이션을 구축하는 것입니다.

## 2. 기능 및 디자인 명세

### 가. 핵심 기능

1.  **지역 선택 UI**:
    *   2단계 드롭다운 메뉴를 제공합니다.
    *   첫 번째 드롭다운에서는 '서울특별시', '경기도', '강원특별자치도' 등 광역자치단체를 선택합니다.
    *   두 번째 드롭다운에서는 첫 번째 선택에 따라 해당 광역자치단체에 속한 기초자치단체(시, 군, 구) 목록이 동적으로 표시됩니다.

2.  **검색어 입력**:
    *   사용자가 '발전시설', '용적률' 등 조례에서 찾고 싶은 특정 키워드를 입력할 수 있는 텍스트 필드를 제공합니다.

3.  **검색 실행 및 리디렉션**:
    *   '검색' 버튼을 클릭하면, 선택된 지역명과 '도시계획 조례' 또는 '군계획 조례', 그리고 입력된 키워드를 조합하여 '자치법규정보시스템'의 상세 검색 URL을 생성합니다.
    *   생성된 URL을 새 탭에서 열어 사용자에게 최종 검색 결과를 보여줍니다. 이를 통해 사용자는 공식 사이트에서 가장 정확한 정보를 얻게 됩니다.

### 나. 디자인 및 스타일

*   **레이아웃**: 화면 중앙에 검색 위젯을 배치하여 사용자가 기능에 집중할 수 있도록 합니다.
*   **색상**: 신뢰감을 주는 파란색 계열을 주 색상으로 사용하고, 명확한 가독성을 위해 어두운 텍스트 색상을 사용합니다.
*   **폰트**: 가독성이 높은 Pretendard 또는 Noto Sans KR 같은 모던한 산세리프 폰트를 사용합니다.
*   **반응형 디자인**: 데스크톱과 모바일 기기 모두에서 최적화된 화면을 볼 수 있도록 반응형으로 디자인합니다.

## 3. 기술 스택

*   **HTML**: 웹페이지의 구조를 정의합니다.
*   **CSS**: 시각적 스타일과 레이아웃을 담당합니다.
*   **JavaScript (ES Modules)**: 동적인 기능(드롭다운 메뉴 업데이트, URL 생성 및 리디렉션)을 구현합니다. 외부 라이브러리 없이 순수 JavaScript로 작성하여 가볍고 빠르게 만듭니다.

## 4. 구현 계획 (초기 버전)

1.  **`blueprint.md` 생성**: 프로젝트의 목표와 계획을 명확히 합니다. (현재 단계)
2.  **`index.html` 작성**: 기본 HTML 구조를 설정합니다. 제목, 설명, 두 개의 드롭다운, 텍스트 입력 필드, 검색 버튼, 그리고 결과 표시를 위한 영역을 포함합니다.
3.  **`style.css` 작성**: 모던하고 직관적인 UI를 위한 CSS 스타일을 적용합니다.
4.  **`main.js` 작성**:
    *   대한민국 행정구역 데이터를 JavaScript 객체 형태로 내장합니다.
    *   페이지 로드 시 첫 번째 드롭다운 (광역자치단체)을 채웁니다.
    *   첫 번째 드롭다운의 선택이 변경될 때마다 두 번째 드롭다운 (기초자치단체)의 내용을 동적으로 업데이트하는 이벤트 리스너를 추가합니다.
    *   '검색' 버튼 클릭 시, 입력된 값들을 조합하여 '자치법규정보시스템' 검색 URL을 생성하고 `window.open`을 통해 새 탭으로 여는 로직을 구현합니다.
